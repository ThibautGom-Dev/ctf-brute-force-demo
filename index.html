⚠️ CYBERSECURITY LESSON & DISCLAIMER

This exercise demonstrates how easily attackers can guess passwords if you base them on personal information (pets, birth years, hobbies).

  • Adding a capital letter and a number at the end (e.g., Buster1990!) DOES NOT make a password strong. Hacking algorithms test these combinations in milliseconds.
  • Defense: Use long passphrases (e.g., BlueHorseRunningFast!) or a Password Manager to generate random strings.
  • Note: This is an educational tool. Never use cracking tools on systems you do not own.
